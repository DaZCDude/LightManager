## About
An open-source, WiZ-compatible light bulb manager for Android.

The app communicates directly with WiZ lights over UDP, so Home Assistant is not required.

This does NOT support managing WiZ lights over the internet. Its supports LAN only.<br>
The app however still needs the network-permission to work.

| Feature                   |  Available |
| -------------             | ------------- |
| Search for lights         | ✅        |
| Manually add light by IP  | ✅        |
| Control on and off state  | ✅        |
| Control brightness        | ✅        |
| Control scene             | ❌        |
| Setup new light           | ❌        |
